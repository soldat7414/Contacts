CONTACTS

Restful web application v1.0.0

developed by Ihor Soldatenko

В приложении используются следующие технологии:

  • Java
  • Spring boot
  • Spring JPA
  • Spring REST
  • Thymeleaf
  • SQLite
  • JQuery
  • HTML
  • CSS
  • Bootstrap


Приложение запускается на порте 8080.
Аутентификация и валидация не прописаны, т.к. на них потребуется больше кода,
чем на само приложение.
